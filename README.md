# moviemanagerapp

Configuração
=====
Abra o arquivo: **app/src/main/res/values/strings.xml** e altera a seguinte linha para definir a 
url onde se encontra servidor que está executando o backend

```xml
    <string name="URL">http://192.168.2.7:8080/api/</string>
```
